# Aviator Multiplier Predictor

এভিয়টর এ কত গুন হবে তা আগে থেকে বুঝতে দিবে

## Overview

Aviator is a project designed to help understand and predict the multiplier values in the Aviator game before each round begins.

## Features

- Real-time multiplier prediction
- Bengali language support
- User-friendly interface
- Predictive analysis

## Installation

```bash
git clone https://github.com/sayemahmedac4-coder/Aviator.git
cd Aviator
```
